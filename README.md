# 📊 Superstore Sales Dashboard (Power BI Project)

## 🎯 Objective
This project analyzes Superstore sales data using Power BI to identify key business insights such as sales performance, profit trends, and regional performance.

---

## 🛠️ Tools Used
- Power BI Desktop
- Excel / CSV (Superstore dataset)
- DAX (Data Analysis Expressions)

---

## 📁 Dataset
- Superstore Sales Dataset (CSV format)
- Includes: Orders, Sales, Profit, Category, Region, Segment, Order Date

---

## 📊 Dashboard Overview

The dashboard contains interactive visuals to analyze business performance:

### 💰 KPI Cards
- Total Sales
- Total Profit
- Total Orders
- Profit Margin

### 📊 Visualizations
- Sales by Category (Bar Chart)
- Sales by Region (Bar Chart / Map)
- Profit by Sub-Category (Bar Chart)
- Sales Trend Over Time (Line Chart)
- Sales by Segment (Pie Chart)

---

## 🧠 Key Insights
- Technology category generates the highest sales
- West region performs better than other regions
- Some sub-categories generate losses despite high sales
- Sales show steady growth with seasonal variations
- Consumer segment contributes the highest revenue
